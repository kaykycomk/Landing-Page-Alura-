# 🚀 Tecboard

Landing page desenvolvida como exercício prático, focada em **estruturação semântica, estilização moderna e responsividade**.

## 📌 Visão geral

O projeto simula a interface de um serviço de monitoramento de aplicações, com foco em uma experiência visual limpa e objetiva.

O layout foi baseado em um protótipo desenvolvido no **Figma**, seguindo fielmente o design proposto.

Apesar de não possuir funcionalidades dinâmicas, o projeto foi construído com boas práticas de front-end, priorizando organização e clareza no código.

---

## 🧠 Principais aprendizados

Durante o desenvolvimento, foram trabalhados conceitos essenciais de front-end:

* Estruturação semântica com HTML5
* Organização de layout com CSS
* Centralização e espaçamento de elementos
* Criação de componentes visuais (botão, hero section)
* Uso de fontes customizadas com `@font-face`
* Estilização com pseudo-classes (`:hover`)
* Aplicação de responsividade com Media Queries
* Interpretação e implementação de layout a partir do Figma

---

## 🎨 Layout e design

* Design centralizado e minimalista
* Baseado em protótipo do Figma
* Destaque visual com uso de cores contrastantes
* Hierarquia clara de informações (título, descrição, CTA)
* Botão com feedback visual ao usuário

---

## 📱 Responsividade

O projeto foi adaptado para diferentes tamanhos de tela:

* 💻 Desktop
* 📱 Tablet
* 📱 Mobile

Utilizando media queries para ajuste de:

* Tipografia
* Espaçamento
* Dimensão de elementos
